# WorldSkills Junior

## Telepítés útmutató

### Frontend


```
cd Frontend
npm install
npm run build
```

### Backend

```
cd Backend
dotnet restore
dotnet build
```

ÉS Visual Studioban a megfelelő gombbal a leépítés is működik.
